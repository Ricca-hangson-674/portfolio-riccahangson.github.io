Technologies

Laravel

un framework web open-source écrit en PHP1 respectant le principe modèle-vue-contrôleur et entièrement développé en programmation orientée objet.

Symfony

un ensemble de composants PHP ainsi qu'un framework MVC libre écrit en PHP. Il fournit des fonctionnalités modulables et adaptables qui permettent de faciliter et d’accélérer le développement d'un site web.

React

une bibliothèque JavaScript dont le but principal est de faciliter la création d'application web monopage, via la création de composants dépendant d'un état et générant une page (ou portion) HTML à chaque changement d'état.

VueJS

un framework JavaScript open-source utilisé pour construire des interfaces utilisateur et des applications web monopages.
